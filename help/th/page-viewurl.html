View URL

View URL เป็น URL ที่ใช้ดูตัวอย่างหน้า แต่ไม่สามารถแก้ไขข้อมูลได้ หากมีการแก้ไขในโปรแกรม เมื่อรีเฟรชหน้าแล้วจะแสดงแก่ผู้ใช้ View URL ทันที

โดยปกติแล้วจะไม่มี View URL มาให้ ซึ่งสามารถสร้างได้โดยการกดปุ่ม reset และหาก URL หลุดรั่วออกไป ใช้ปุ่ม reset เพื่อเปลี่ยน URL ได้

วัตถุประสงรค์

  • ทดสอบหน้า
  • ทดสอบหน้าบนอุปกรณ์อื่นๆ เช่น ไอแพด
  • ตรวจสอบคุณภาพ
  • เป็นตัวอย่างหนังสือ
  • Kyou จะแนะนำให้สมาชิกโครงการใช้ view URL เมื่อพยายามเข้าถึงหน้าที่ล็อคอยู่

หมายเหตุ

ผู้เข้า view URL สามารถเห็นโค้ดของหน้านั้นในรูปแบบ Kyou ได้ (แต่ไม่สามารถแก้ไขได้)
