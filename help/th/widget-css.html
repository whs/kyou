วิดเจดท์ CSS

วิดเจดท์ CSS ใช้ในการแต่งหน้า โครงสร้างหน้าบางตัวอาจไม่รองรับตัวเลือกบางตัว

Name
ชื่อของวิดเจดท์ในโปรแกรมแก้ไข (ไม่ใช้ในผลลัพท์)
Text color
สีข้อความ เลือกจากจานสี หรือพิมพ์สี CSS เอง (เช่นชื่อสี cornflowerblue หรือสีระบบ rgb())
Background color
สีพื้นหลัง ป้อนสี CSS เองได้เช่นกัน
Background fix
  • Fixed ทำให้ภาพพื้นหลังติดอยู่ที่เดิมขณะเลื่อนเนื้อหา
  • Move with scrolling เลื่อนภาพพื้นหลังไปขณะเลื่อนเนื้อหา
Background tiling
หากขนาดภาพพื้นหลังเล็กกว่าขนาดของหน้า ใช้ตัวเลือกนี้เพื่อเรียงภาพต่อกันแทนที่จะเติมพื้นที่เหลือด้วยสีพื้นหลัง
Background image
URL ภาพพื้นหลัง
Background resize
  • No ใช้ตามตัวเลือก Background fix
  • Cover ปรับขนาดภาพพื้นหลังถ้าเล็กกว่าหน้าให้พอดีหน้า โดยรักษาอัตราส่วนไว้ (ภาพที่ปรับขนาดแล้วอาจมีขนาดใหญ่กว่าหน้า)
  • Contain ปรับขนาดภาพพื้นหลังให้ขนาดเท่าหน้าโดยไม่สนอัตราส่วน
Background position
วางจุดกำเนิดภาพพื้นหลังไว้ที่จุดนี้
บนเยื้องซ้าย บนจัดกลาง บนเยื้องขวา
กึ่งกลางซ้าย กึ่งกลางจัดกลาง กึ่งกลางเยื้องขวา
ล่างเยื้องซ้าย ล่างจัดกลาง ล่างเยื้องขวา
Font size
ขนาดแบบอักษร หน่วยเป็นพอยต์ (pt)
Custom CSS
ใส่ CSS เองเหมือนที่ใช้ในแท็ก <style> เว้นว่างไว้หากไม่ทราบว่าใช้อย่างไร
