Version control

Kyou จะบันทึกการแก้ไขทั้งหมดโดยอัตโนมัติ แต่บางครั้งอาจจะจำเป็นต้องย้อนกลับไปยังเอกสารรุ่นเก่า

ระบบควบคุมรุ่นช่วยให้สามารถบันทึกสถานะของหน้า และย้อนกลับไปยังสถานะใดๆ ที่เคยบันทึกไว้ก่อนหน้านี้

Create revision

เมนู Create revision ใช้บันทึกสถานะของหน้าในปัจจุบัน และสามารถเพิ่มเติมข้อความกำกับได้

Run version control

เมื่อคลิกปุ่ม "Run version control" จะปรากฎหน้าใหม่ขึ้นมา ซึ่งในหน้านี้จะแสดงสถานะที่บันทึกไว้ล่าสุด พร้อมทั้งคำอธิบายและเวลาที่บันทึกในกล่องด้านล่าง

สามารถคัดลอกข้อความ และเลื่อนหน้าต่างตัวอย่างหน้าได้

  • เลื่อนไปยังรุ่นเก่ากว่า
  • เลื่อนไปยังรุ่นใหม่กว่า
  • กลับไปใช้รุ่นนี้
  • กลับไปที่โปรแกรม
คำเตือน: ไม่สามารถยกเลิกการย้อนกลับไปใช้รุ่นใดๆ ได้ หากต้องการจะย้อนกลับมายังรุ่นปัจจุบันในอนาคต ให้บันทึกสถานะของหน้าปัจจุบันไว้ก่อน
