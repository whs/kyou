ภาพโต้ตอบ

ระบบภาพโต้ตอบ เพิ่มความมีชีวิตให้กับภาพนิ่งหรือภาพเคลื่อนไหว

การเลือกภาพ

ในหน้าแรกโปรแกรมจะแสดงรายการภาพที่ได้อัพโหลดไว้ในตัวจัดการทรัพยากร คลิกเลือกภาพใดๆ แล้วกดปุ่ม Create เพื่อเรียกข้อมูลที่บันทึกไว้ หรือสร้างภาพใหม่ นอกจากนี้ยังสามารถใช้ปุ่ม Open เพื่อแสดงรายการภาพที่มีข้อมูลการโต้ตอบอยู่แล้ว

การใช้โปรแกรมแก้ไขภาพโต้ตอบ

  1. คลิกปุ่ม เพื่อเพิ่มจุดโต้ตอบ
  2. ย้ายกล่องสีที่ปรากฏทางมุมบนซ้ายของภาพ ไปยังตำแหน่งที่ต้องการ แล้วปรับขนาดให้เหมาะสม
  3. คลิกที่กล่องสี จะปรากฎเมนูในแถบข้าง และกล่องจะสีเข้มขึ้น
  4. ตั้งค่า
  5. คลิก Save

ใช้ปุ่ม เพื่อดูตัวอย่างภาพโต้ตอบที่จะปรากฎในหน้าหนังสือ

ตัวเลือกภาพโต้ตอบ

Hit icon
ไอคอนที่จะปรากฎที่ศูนย์กลางของกล่อง เพื่อแสดงว่านี่คือจุดโต้ตอบ (ไม่จำเป็นต้องใส่)

Change background

ตัวเลือกนี้จะเปลี่ยนภาพโต้ตอบเป็นภาพใหม่เมื่อมีการชี้เมาส์

New image
URL ของภาพใหม่ โดยจะต้องมีขนาดเท่ากับภาพเดิม

Display image

ตัวเลือกนี้จะแสดงภาพอื่นมาทับภาพเดิมเมื่อมีการชี้เมาส์

Image
URL ของภาพ เมื่อบันทึกแล้วจะปรากฎภาพทางซ้ายบน สามารถลากภาพไปยังตำแหน่งอื่นได้ตามต้องการ

Display text

ตัวเลือกนี้จะแสดงข้อความบนภาพเมื่อมีการชี้เมาส์

Text
ใส่ตัวหนังสือที่จะปรากฎในกล่อง สามารถใช้ HTML ได้ สามารถลากและปรับขนาดกล่องได้ โดยปรับขนาดได้เฉพาะตามแนวนอนเท่านั้น และความสูงจะคำนวณให้โดยอัตโนมัติ

Link

เมื่อคลิกที่เป้าหมายแล้วจะเปลี่ยนไปยังหน้าหรือเว็บเพจอื่น

Link
URL ของหน้าใหม่ ใช้ปุ่ม Pick เพื่อเลือกหน้าหนังสืออื่น

เพิ่มภาพโต้ตอบในหนังสือ

การเพิ่มภาพโต้ตอบ ทำได้โดยเพิ่มวิดเจดท์ชื่อ "Interactive Image" แล้วเลือกชื่อไฟล์จากในรายการ
