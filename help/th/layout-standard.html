โครงสร้างมาตรฐาน

โครงสร้างมาตรฐานจะเป็นโครงสร้างที่ถูกใช้ในหน้าส่วนมากในหนังสือ แม้จะสามารถปรับค่าได้น้อย แต่มันคือโครงสร้างที่ทรงพลังที่สุด

โครงสร้างมาตรฐานออกแบบมาให้ใช้กับ "วิดเจดท์" หรือโค้ดเล็กๆ น้อยๆ ที่มาประติดประต่อรวมกันกลายเป็นหน้า การเพิ่มวิดเจดท์ทำได้โดยการกดปุ่มเพิ่มในเมนูด้านข้าง

การตั้งค่า

Transparent BG
ซ่อนสีพื้นหน้า เหมาะสำหรับการเน้นภาพพื้นหลังให้โดดเด่น
Dark layout
ใช้โทนสีมืด เป็นอีกทางเลือกนอกจากการปรับสีด้วยตัวเอง
Front color
สีพื้นหน้า

คำแนะนำ

โครงสร้างนี้ถูกออกแบบให้สามารถตกแต่งได้จากวิดเจดท์ CSS โดยสามารถปรับสีตัวอักษร สีพื้นหลัง ภาพพื้นหลังและอื่นๆ ได้
