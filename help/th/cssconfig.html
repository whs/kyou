CSS

วิดเจดท์บางตัวอาจปิดการใช้บางตัวเลือก

Float
ตำแหน่งการวางวิดเจดท์ ใช้ปุ่ม เพื่อกลับไปยังค่าปกติ
Text Align
การจัดตัวหนังสือภายในวิดเจดท์
Clear
ทำให้พื้นที่ด้านที่เลือกไว้ของวิดเจดท์ปราศจากวิดเจดท์ที่กำลัง Float อยู่
Font size
ขนาดแบบอักษร หน่วยเป็นพอยต์ (pt)
Size (px)
ขนาดกว้าง x สูงภายในของวิดเจดท์ โดยขนาดนี้จะไม่รวม padding หรือ margin ใดๆ วิดเจดท์ส่วนมากสามารถระบุขนาดได้โดยการกำหนดความกว้างหรือความสูงเพียงค่าเดียว หรือกำหนดทั้งสองค่าก็ได้
Text color
สีข้อความ เลือกจากจานสี หรือพิมพ์สี CSS เอง (เช่นชื่อสี cornflowerblue หรือสีระบบ rgb())
Background color
สีพื้นหลัง ป้อนสี CSS เองได้เช่นกัน
Margin
ช่องไฟระหว่างวิดเจดท์ ในพื้นที่นี้จะไม่มีการใส่พื้นหลัง
Padding
ช่องว่างระหว่างขอบวิดเจดท์และเนื้อหา ในพื้นที่นี้จะมีการใส่พื้นหลัง
Background image
URL ภาพพื้นหลัง
Background tiling
หากขนาดภาพพื้นหลังเล็กกว่าขนาดของหน้า ใช้ตัวเลือกนี้เพื่อเรียงภาพต่อกันแทนที่จะเติมพื้นที่เหลือด้วยสีพื้นหลัง
Background position
วางจุดกำเนิดภาพพื้นหลังไว้ที่จุดนี้
Top left Top center Top right
Middle left Middle center Middle right
Bottom left Bottom center Bottom right
Custom CSS
ใส่ CSS เองเหมือนที่ใช้ใน attribute style (ไม่ใช่ tag) เว้นว่างไว้หากไม่ทราบว่าใช้อย่างไร
