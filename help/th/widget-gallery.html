คลังภาพ

วิดเจดท์ "Gallery" ใช้ในการแสดงภาพชุด โดยมี 2 ลักษณะ คือ แบบ Slider และแบบ Images

Slider

รูปแบบ slider ใช้เนื้อที่น้อย แต่สามารถแสดงภาพได้ทีละภาพ โดยทุกภาพควรมีอัตราส่วนเท่ากัน

Image Size
ขนาดกว้างxสูงของตัวเลื่อนภาพ โดยภาพจะถูกย่อขนาดลงให้อยู่ภายในกล่องนี้ และขนาดของกล่องจะรวมถึงพื้นที่ที่ใช้ในการแสดงภาพย่อ หากเปิดให้แสดง
Transition
เอฟเฟกท์การเปลี่ยนภาพ
  • Random: สุ่มเลือกจากรายการต่อไปนี้
  • Cube: แผ่นภาพเป็นด้านหนึ่งของกล่องสามมิติ กล่องจะถูกหมุนไปทางซ้าย (horizontal) หรือหมุนลง (vertical) เบราว์เซอร์จะต้องรองรับ 3D CSS มิเช่นนั้นจะไม่มีเอฟเฟกท์
  • Fade: เฟดจางหายระหว่างภาพ
  • Slice: ตัดภาพเป็นชิ้นๆ ตามแนวซ้ายไปขวา (horizontal) หรือจากล่างขึ้นบน (vertical) แล้วแต่ละชิ้นจะร่วงตามๆ กันไปหาจุดกำเนิดการตัด
  • Slide: ภาพเลื่อนไปทางซ้าย (horizontal) หรือล่าง (vertical)
  • Scale: ภาพขยายใหญ่ขึ้น และเฟดออก
  • Block Scale: ตัดภาพเป็นหลายๆ กล่อง และแต่ละกล่องลดขนาดลง
  • Kaleidoscope: ตัดภาพเป็นหลายๆ กล่อง และแต่ละกล่องค่อยๆ เริ่มจางหายไป เริ่มจากกล่องทางซ้ายบนสุดมาเรื่อยๆ ถึงขวาล่างสุด
  • Fan: ตัดภาพเป็นหลายๆ กล่องจากซ้ายไปขวา แต่ละชิ้นร่วงไปที่ขอบขวาล่าง
  • Blind: ตัดภาพเป็นหลายๆ ชิ้นจากบนลงล่าง (horizontal) หรือซ้ายไปขวา (vertical) แต่ละชิ้นลดขนาดลงและเฟดออก
เอฟเฟกท์บางตัวอาจแสดงผลไม่ถูกต้องในหน้าต่างแสดงตัวอย่าง เนื่องจากข้อจำกัดทางเทคนิค
Effect duration
ความยาวของเอฟเฟกท์เปลี่ยนภาพเป็นมิลลิวินาที (1000ms = 1 วินาที)
Slide duration
ระยะเวลาแสดงแต่ละภาพเป็นมิลลิวินาที ใส่เลข 0 เพื่อปิดการเลื่อนภาพอัตโนมัติ
Thumbnails
ทำเครื่องหมายเพื่ออนุญาตให้เลื่อนภาพเองได้ โดยมีภาพตัวอย่างของทุกภาพให้คลิก

Images

รูปแบบ images เป็นรูปแบบพื้นฐานที่สุด โดยแสดงภาพทุกภาพพร้อมกัน และทุกภาพจะต้องมีขนาดเดียวกัน

Image size
ขนาดของแต่ละภาพ โดยทุกภาพจะใช้ขนาดเดียวกันหมดไม่ว่าขนาดหรืออัตราส่วนต้นฉบับจะเป็นอย่างไร
Margin between images
ช่องไฟระหว่างภาพ
Hover to zoom
หากเลือกไว้ เมื่อผู้อ่านเลื่อนเมาส์หรือแตะบนภาพ ภาพจะถูกขยาย
