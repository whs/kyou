วิดเจดท์ WYSIWYG

WYSIWYG (What You See Is What You Get - สิ่งที่คุณเห็นคือสิ่งที่คุณได้รับ) เป็นวิธีที่ง่ายที่สุดในการจัดวางเนื้อหา

วิธีที่ง่ายที่สุดในการใช้คือการคัดลอกเนื้อหาจากโปรแกรม Microsoft® Word™ หรือโปรแกรมประมวลผลคำอื่นๆ แล้ววางลงโดยตรง ตัววิดเจดท์จะจัดรูปแบบให้โดยอัตโนมัติ

คลิกปุ่ม เพื่อบันทึกงานหลังแก้ไขแล้วเสมอ

แถบเครื่องมือ

  • ตัวหนา: เปิดปิดการใช้ฟอนต์หนากับข้อความที่เลือกไว้
  • ตัวเอียง: เปิดปิดการใช้ฟอนต์เอียงกับข้อความที่เลือกไว้
  • ขีดเส้นใต้: เปิดปิดการขีดเส้นใต้ข้อความที่เลือกไว้
  • ขีดฆ่า: เปิดปิดการขีดฆ่าข้อความที่เลือกไว้
  • ตัวห้อย: เปิดปิดการทำให้ข้อความที่เลือกไว้เป็นตัวห้อย
  • ตัวยก: เปิดปิดการทำให้ข้อความที่เลือกไว้เป็นตัวยก
  • ล้างการจัดรูปแบบ: ยกเลิกการจัดรูปแบบข้อความที่เลือกไว้ทุกชนิด
  • รายการตัวเลข: เพิ่มรายการตัวเลข กดแป้น enter เพื่อขึ้นข้อต่อไป และกดสองครั้งเพื่อสิ้นสุดรายการ
  • รายการจุด: เพิ่มรายการจุด กดแป้น enter เพื่อขึ้นข้อต่อไป และกดสองครั้งเพื่อสิ้นสุดรายการ
  • ลิงก์: ทำให้ข้อความที่เลือกไว้เป็นการเชื่อมโยง
  • เลิกลิงก์: ยกเลิกการเชื่อมโยงกับข้อความที่เลือกไว้ สามารถกดใช้ได้เฉพาะเมื่อมีการเลือกข้อความที่เป็นการเชื่อมโยง
  • สีข้อความ: เปลี่ยนสีข้คอวามที่เลือกไว้
  • สีพื้น: เปลี่ยนสีพื้นหลังข้อความที่เลือกไว้ ไม่แนะนำ
  • จัดซ้าย: จัดข้อความที่เลือกไว้ไปทางซ้าย
  • จัดกลาง: จัดข้อความที่เลือกไว้อยู่ตรงกลาง
  • จัดขวา: จัดข้อความที่เลือกไว้ไปทางขวา
Normal
เลือกชนิดของย่อหน้า เช่นระดับหัวเรื่อง
Font
เลือกแบบอักษรของข้อความที่เลือกไว้ ไม่แนะนำ
Size
ปรับขนาดแบบอักษรของข้อความที่เลือกไว้ หากจะจัดข้อความให้เป็นหัวเรื่อง ควรใช้ตัวเลือกชนิดย่อหน้าแทนการขยายแบบอักษร
  • ภาพ: แทรกรูปภาพ
  • ตาราง: แทรกตาราง
  • อักขระพิเศษ: เพิ่มอักขระพิเศษ​เช่น © ® •
  • เส้นคั่น: เพิ่มเส้นแบ่งแนวนอน
  • ค้นหา: ค้นหาข้อความ
  • แทนที่: ค้นหาและแทนที่้ขอความ
  • เต็มจอ: ขยายขนาดตัวแก้ไขให้เต็มจอ
  • Source: ดูและแก้ไขต้นฉบับ HTML
  • Help: เปิดหน้าจอนี้
  • บันทึก: บันทึกการแก้ไขหน้า
  • ซ่อน: ซ่อนแถบเครื่องมือ

เพิ่มลิงก์

Link Type
ชนิด URL จะลิงก์ไปยังหน้าอื่นในอินเทอร์เน็ต หรือในหนังสือ ไม่สนับสนุนการใช้ลิงก์ชนิดอื่น
Protocol
คำนำหน้า URL เช่น http://, https://
URL
ปลายทางลิงก์​ โดยไม่ต้องระบุโปรโตคอลนำหน้า เช่น sunburn.in.th.
Browse Server
คลิกปุ่มนี้เพื่อลิงก์ไปยังแฟ้มที่อัพโหลดไว้
Pages
เลือกชื่อหน้าในรายการนี้เพื่อลิงก์ไปยังหน้านั้นๆ
Target (แท็บ)
เปิดลิงก์ที่ใด
  • <not set>: เปิดลิงก์ในหน้านี้
  • <popup window>: เปิดลิงก์ในหน้าต่างใหม่
  • New Window (_blank): เปิดลิงก์ในแท็บใหม่

แทรกรูปภาพ

URL
URL เต็มของภาพ รวมทั้งโปรโตคอลนำหน้า
Browse Server
คลิกปุ่มนี้เพื่อแทรกภาพจากแฟ้มที่อัพโหลดไว้
Width
ความกว้างของภาพ
Height
ความสูงของภาพ คลิกปุ่มแม่กุญแจเพื่อปลดการล็อคอัตราส่วน
Alignment
ตำแหน่งวางภาพ โดย <not set> จะวางภาพในบรรทัด
Link (แท็บ)
เพิ่มคลิกลิงก์ให้กับภาพ โดยการตั้งค่าคล้ายกับหน้าต่างเพิ่มลิงก์
