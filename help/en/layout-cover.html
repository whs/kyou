The cover layout

Cover is unarguably the most important part of any book. The standard Kyou cover was designed for a magazine, CoreAnime. It contains 6 tiles on the left side, an optional ads tile, a featured content, magazine logo, issue number and Facebook widget

Cover image

Cover image can be a list of images, an image or a MP4 video (detected by file's extension)

Note that you may not add more than one video file or use other video formats than MP4.

Demo: Scrolling images is used in CoreAnime 1, static image is used in CoreAnime 2, and video cover is used in CoreAnime 3.

Configurations

Logo
Logo file. Must be an image. The height is 150px and the width should be less than 400px. The layout automatically resizes the image file to fits automatically.
Facebook page
Optional Facebook page short URL. For example, if your page is accessible from https://www.facebook.com/coreanime enter "coreanime". The layout will show a like box on the cover
Issue
Issue number such as "Issue 2". Any text is okay.
Issue Name
Optional Issue name such as "Spring 2012". Any text is okay.
Featured Content Image
Optional Image to show above the logo. Should be a transparent image but a square one is okay.
Featured Content Link
Optional Link of the featured content image. Must be entered if an image is present. If there're no featured content the logo will appear on top right.
Zoom cover items
If checked, cover tiles and featured content tile could be enlarged by hover.
Background transition
Only take effects when there're multiple cover images
  • Fade Crossfade images
  • Pan & Fade Pan images down and crossfade. Also known as Ken Burns effect

Cover tiles

Cover tiles are actually a widget called "Item" and can be added in the new widget box. The optimum number of tiles supported is 6 plus one additional ads tile.
