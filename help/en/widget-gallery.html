Gallery widget

Gallery widget is used to show a set of images. It have two defined styles, "Slider" and "Images"

Slider

The slider style occupy less space but only display one images at a time. All images should have the same dimension

Image Size
The width x height of the slider. Images will be downsized. This includes the space occupied by the thumbnails if enabled.
Transition
The effect used when changing between images
  • Random: One of these will be chosen at random.
  • Cube: The slide is on a 3D box and will be rotated left (horizontal) or down (vertical). Requires 3D CSS support in browser or the effect will not be shown.
  • Fade: Crossfade between images.
  • Slice: The image is sliced to several pieces from left to right (horizontal) or bottom to top (vertical) and the pieces are falling apart to the origin of the cut.
  • Slide: The image moves to left (horizontal) or bottom (vertical).
  • Scale: The image is zoomed and faded out
  • Block Scale: The image is sliced to several blocks and the blocks reduce in size.
  • Kaleidoscope: The image is sliced to several blocks and the blocks fade out from top left to bottom right
  • Fan: The image is cut to several pieces from left to right and each pieces fall to bottom right
  • Blind: The image is cut to several pieces from top to bottom (horizontal) or left to right (vertical) and each pieces reduce in size and fade out.
Some transition may render incorrectly in preview due to technical limitations
Effect duration
How long does the transition takes in milliseconds. (1000ms = 1sec)
Slide duration
How long does an image is shown in milliseconds. Enter 0 to disable auto advancing.
Thumbnails
Check to enable manual skipping by showing thumbnails of all images

Images

The images style is the most basic style. It show all images at the same time. All images must have the same size.

Image size
The size of each images. All image will use this size regardless of the original size or dimension.
Margin between images
Spacing between images.
Hover to zoom
If enabled, when the reader move mouse or touch on the image it will be enlarged
