CSS

Some options may not be available in some widgets

Float
Align of the widget. Use the to return to normal placing.
Text Align
Align of text inside the widget
Clear
This option make room in the selected side of widget void of floating widgets.
Font size
Base font size, in points.
Size (px)
Width x Height of widget's internal size in pixels. The internal size specified here does not include any padding or margin. Most widget's size could be specified by only width or height, or both.
Text color
Text color. Use the color picker or type your own CSS color (such as cornflowerblue, or rgb() color)
Background color
Background color. Manual CSS color is OK.
Margin
Spacing between widgets. Background will not be placed in margin area.
Padding
Spacing between widget's border and content. Background will be placed in padding area.
Background image
URL of background image
Background tiling
If the background image's size is smaller than page's size, enable this option to tile the image instead of filling with background color
Background position
Place the background image origin in this location:
Top left Top center Top right
Middle left Middle center Middle right
Bottom left Bottom center Bottom right
Custom CSS
Enter custom CSS as you would use in a style attribute (not tag) here. Leave blank if you don't know what you're doing.
