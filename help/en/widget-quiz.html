Quiz widget

Quiz widget is used for self reviewing questions. Readers can answer multiple choices questions and get scored. They also can retry unlimited times.

To save changes, click on any question and the save button will appear.

Quiz answer is clearly visible in page's source. This widget is designed for self-reviewing question, not an exam.

Types

  • Full list of questions

    Show all questions at once

  • Show one question at a time

    Show only one question. Reader must use back and forward button to navigate. The last page always a grading button.

Adding question

  1. Click the button in the top list to add a question
  2. Click on Enter question
  3. Enter question in the question box
  4. Click the button in the bottom list a few times until enough answer boxes are shown.
  5. Enter each answers in answer boxes
  6. For each correct answers, enter score the reader will receive in the score box (such as 1 for correct, and 0 for incorrect). A question can have multiple correct choices and each choice could give different scores.
  7. Click Save.

Tips

Use the size CSS option to adjust widget's size
