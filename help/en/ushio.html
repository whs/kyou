Version control

Kyou automatically saves all changes. However, sometimes you might need to revert some changes to an older point in time.

The version control system does this by allowing you to save a page's state and allows you to return to any previously saved state.

Create revision

The create revision menu is use to save page's current state. You also can optionally enter a description.

Run version control

After clicking "Run version control", you will be taken to a new page. In this page the latest saved state is shown. The description and the time that this revision was saved is shown in the box below

Note that the preview content can be scrolled and copied from.

  • Move backwards in time
  • Move forward in time
  • Return to this revision
  • Return to editor
Warning: Returning to a revision cannot be undone. Save revision first if you need to return to the current revision in the future.

Note

Version control requires a WebGL compatible browser and graphics card.
