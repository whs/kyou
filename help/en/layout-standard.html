The standard layout

Standard layout is the layout you will be using in most pages of your book. It have few configurable options, but it is the most powerful layout of all.

The standard layout is designed to use with "widgets", tiny bits of code that tied together to create a page. You can add a widget by pressing the add button on the sidebar.

Configuration

Transparent BG
Disable the front color. Useful when you want to show your background image
Dark layout
Use the dark color scheme. Alternate to manually choosing color.
Front color
The page's foreground color

Tips

The standard layout is designed to be styled with the CSS widget. Text color, background color, background image and many other options can be changed from CSS widget.
