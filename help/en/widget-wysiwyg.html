WYSIWYG widget

WYSIWYG (What You See Is What You Get) is the easiest way to format your contents.

The easiest way to use it is to copy your contents from Microsoft® Word™ or other word processors and paste it directly. The widget will format it automatically

Always click to save your work after editing

Toolbar

  • Bold: Toggle bold font to the selected text.
  • Italic: Toggle italic font to the selected text.
  • Underline: Toggle text underlining to the selected text.
  • Strike: Toggle text strike through to the selected text.
  • Subscript: Toggle text subscript to the selected text.
  • Superscript: Toggle text superscript to the selected text.
  • Remove formatting: Cancels all text formatting to the selected text.
  • Numbered list: Add a numbered list. Press enter to add next point, double enter to end listing.
  • Bulleted List: Add a bulleted list. Press enter to add next bullet, double enter to end listing.
  • Link: Make selected text a link.
  • Unlink: Unlink selected text. Only usable when selecting linked text.
  • Text color: Change selected text's font color.
  • Background color: Change selected text's background color. not recommended
  • Justify left: Make selected text left-aligned.
  • Justify center: Make selected text center-aligned.
  • Justify right: Make selected text right-aligned.
Normal
Set paragraph's type such as heading level
Font
Set selected text's font face. not recommended
Size
Set selected text's font size. If this text is a header, paragraph type should be set instead of enlarging font size
  • Image: Add an image
  • Table: Add a table
  • Special Character: Add a special character such as © ® •.
  • Horizontal Rule: Add a horizontal divider.
  • Find: Find text.
  • Replace: Find and replace text.
  • Maximize: Enlarges the editor to full screen.
  • Source: View and edit HTML source directly.
  • Help: Open this page.
  • Save: Commit changes to the page.
  • Collapse: Hide the toolbar

Add Link

Link Type
URL links to other pages on the internet or in the same book. Other type of links are not supported.
Protocol
URL prefix such as http://, https://
URL
Link target without the protocol prefix such as sunburn.in.th.
Browse Server
Click this button to browse uploaded files and add link to it.
Pages
Select page name in the dropdown to link to that page.
Target (tab)
Where to open the link
  • <not set>: Open link in same page
  • <popup window>: Open link in new popup window
  • New Window (_blank): Open link in new tab

Add Image

URL
Full URL of the image including protocol prefix.
Browse Server
Click this button to add image from uploaded file.
Width
Image's width
Height
Image's height. Click the padlock button to unlock dimension ratio.
Alignment
Where to place the image. <not set> places the image inline.
Link (tab)
Add clickable link to image. The settings are similar to the add link window.
