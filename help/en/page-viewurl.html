View URL

View URL is a read-only URL used to preview the page. Every changes in the editor will take effect in the preview after a refresh.

By default there're no view URL for a page and requires a reset to create one. If the URL is leaked, use the reset button to change it.

Common use cases

  • Testing your page
  • Testing on mobile devices, such as the iPad
  • QA
  • As a teaser of the book
  • Kyou will offer to use view URL to project contributors if a page is locked

Note

User of view URL can see the source of the page in Kyou format. (But may not change it)
