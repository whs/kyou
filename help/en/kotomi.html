Interactive Image

Interactive image adds interactiveness to your static or animated images.

Selecting Images

The front page will show list of images you have uploaded in the resource manager. Click on an image and click Create to load the saved data or create new image. Or click the Open button and a list of images that have interactive data will be shown.

Using interactive image editor

  1. Click button to add new interactive spot
  2. Move the colored box that appear on top left of the image to desired spot and resize it to desired size.
  3. Click on the colored box. A menu will appear on the sidebar and the box will turns darker
  4. Set options
  5. Click Save.

Use the button to preview the interactive image as it will appear in book pages.

Interactive options

Hit icon
Icon that will appear in the center of the box to indicate that this is interactive target. (Optional)

Change background

This option changes the interactive image to a new image on hovering

New image
URL of the new image. Must have the same size with current one

Display image

This option show another image on top of current one on hovering

Image
URL of the image. After saving, it will appear in top left. You then can drag to reposition the image.

Display text

This option show a popup text on top of the image on hovering

Text
Enter the text inside of popup. HTML is OK. You can drag and resize the popup. Resizing works horizontally only and the height will be automatically calculated.

Link

On clicking the target, navigates to other page or webpage.

Link
URL of new page. Use the Pick button to pick other book pages.

Adding interactive image to book

To add an interactive image, add a widget "Interactive Image" and pick file name from the list
