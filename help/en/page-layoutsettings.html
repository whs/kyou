Page settings

Name
The page's name. This will be used as reader-visible page name and in the navigator.
Stage
Progress of the page's creation. Will be show only in editor. This field's value does not affect any operation of the program and serve only as a note to project's contributors.
  • Planning: Page structure is under planning.
  • Waiting for content: Content is being written elsewhere.
  • Working: Page layout and content is being placed
  • QA: Page is done. Requires QA staff to verify the page
  • Done: Page can be shipped
Layout
Layout affects how the page is shown to the user. Also, it affects what kind of widgets could be placed on the page.

Layouts

Kyou included a few layouts. Some additional templates can be made available or some template mentioned here may not be available.

  • Cover

    Book's cover. User can add up to 6 boxes to the left side. The page's background is a scrolling image gallery or a video

  • Full page slider

    An image gallery that reader can use up/down arrow keys to navigate. Image can optionally be described with a text.

  • Standard

    Standard book page. Most widgets only works in this layout.
