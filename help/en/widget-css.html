CSS Widget

CSS widget is used for styling of the page. Some layout may not support some options.

Name
Name of the widget in editor. (Not used in output)
Text color
Text color. Use the color picker or type your own CSS color (such as cornflowerblue, or rgb() color)
Background color
Background color. Manual CSS color is OK.
Background fix
  • Fixed make the background image locks in place while scrolling.
  • Move with scrolling scrolls the background image while scrolling.
Background tiling
If the background image's size is smaller than page's size, enable this option to tile the image instead of filling with background color
Background image
URL of background image
Background resize
  • No Obeys background fix option
  • Cover Resizes the background image if it is smaller than the page to fill the page and constrains ratio (the resized background could be larger than the page)
  • Contain Resizes the background image to page's size regardless of image ratio.
Background position
Place the background image origin in this location:
Top left Top center Top right
Middle left Middle center Middle right
Bottom left Bottom center Bottom right
Font size
Base font size, in points.
Custom CSS
Enter custom CSS as you would use in a <style> tag here. Leave blank if you don't know what you're doing.
